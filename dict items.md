# Dictionary items

what they are:
The items() method returns a view object.

when you would use them:
When you want the key-value pairs of the dictionary, as tuples in a list.


* advantage 1
* advantage 2

how to code them:

```python
dict = {'Name': 'Zara', 'Age': 7}
print "Value : %s" %  dict.items()
[('Age', 7), ('Name', 'Zara')]
```
